# metaplot
 
<!-- badges: start -->

Visualize m6A peaks distribution on RNA.

<!-- badges: end -->

# usage
```R
source("https://raw.githubusercontent.com/yang-dongxu/metaplot/master/metagene.R")
```

Then use functions to process:
- `prepare_features`
- `calculate_relative_position`
